# CRM con Django

## Descripción
Este es un sistema de gestión de relaciones con clientes (CRM) desarrollado con Django. El sistema permite a los usuarios registrarse, iniciar sesión y gestionar clientes: agregar, eliminar, actualizar y visualizar su información.

## Funcionalidades
- Registro de usuarios
- Inicio de sesión y cierre de sesión
- Gestión de clientes:
  - Agregar nuevos clientes
  - Eliminar clientes existentes
  - Actualizar información de clientes
  - Visualizar la lista de clientes con detalles

## Tecnologías Utilizadas
- **Lenguaje de programación:** Python
- **Framework:** Django
- **Base de datos:** SQLite 

## Instalación

Para instalar y ejecutar este proyecto, sigue los pasos a continuación:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/EzequielPaz/CRM.git
   cd crmproject

2. **Instala las dependencias:**
  pip install -r requirements.txt

3. **Realiza las migraciones de la base de datos:**
  python manage.py migrate

4. **Inicia el servidor de desarrollo:**
  python manage.py runserver
